CircularProgressIndicator

As the name suggests, CircularProgressIndicator widget is used in flutter to create Circular Progress Indicators.

There are 2 types of progress bars: determinant and indeterminant. Determinant progress bars are those which shows the absolute progress of the task and provides an idea when the task is going to complete, whereas indetermint progress bars does not show the absolute progress of the task as it is not known when the task is going to complete.

Both Determinant and Indeterminant Circular bar can be created in flutter with CircularProgressIndicator widget. If you provide the value parameter, the progress bar becomes a determinant otherwise its indeterminant. The value parameter can take value ranging from 0.0 (0% completion) to 1.0 (100% completion)

Indeterminant CircularProgressIndicator:

CircularProgressIndicator(),

Determinant CircularProgressIndicator:

CircularProgressIndicator( value = 0.4), // for 40% completion

Notable parameters

  • backgroundColor
  • valueColor
  • strokeWidth
  • value

backgroundColor

This parameter is used to specify the background color of the progress indicator. The default background color is the color of the page.

valueColor

This parameter is used to specify the value color of the progress indicator in terms of AlwaysStoppedAnimation. In case of determinant progress bar, this color will fill the completed part of the progress indicator.

strokeWidth

This parameter is used to specify the width of the progress indicators.

value

This parameter is used only for the Determinant type of progress indicators. Use this value to show the percentage of completion. It can take value from 0.0 (0% completion) to 1.0 (100% completion).

Code snippet with all the above parameters:

CircularProgressIndicator( backgroundColor: Colors.redAccent, valueColor: AlwaysStoppedAnimation(Colors.grey), strokeWidth: 5, value = 0.4 // showing 40% completion (remove this parameter if its inditerminant progress indicator) )

If you want to change the default size of CircularProgressIndicator, you can wrap the same inside a SizedBox widget specifying the width and height parameter for the required size, something like this:

SizedBox( width: 100.0, height: 100.0, child: CircularProgressIndicator( backgroundColor: Colors.grey, value: 0.8, strokeWidth: 5, )));
