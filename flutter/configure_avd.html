Configure Android Virtual Machine

To run android application in your desktop or laptop, you need to configure AVD (Android Virtual Machine). The steps given below shows the steps for creating AVD in Android Studio

.

Steps:

  1. Start Android Studio
  2. Click on Tools > AVD Manager
  3. Click the button Create Virtual Device. You should see the following screen:

  4. On the left side, Phone should already be selected. Select one of the device like Pixel 2 and click NEXT
  5. Select one of the System image, for example, R. If there is a DOWNLOAD option, click on download and let the download get over. Once the download is over, click NEXT. If there is no download option for the selected image, just click NEXT

  6. Just give a name for your AVD, or keep the default name and click FINISH

  7. Your AVD will take some time to create, once done, you will see the following screen, showing you the list of all AVDs created.

  8. To start the AVD, click on the green PLAY button and wait till the AVD is up and running. It might take quite sometime for the AVD to be up and running
  9. Once your AVD has started, you will see a screen like this, depending on the configuration of the AVD and the System Image selected.

  10. Your running AVD will appear in the android studio toolbar like the following:

Now you are ready to run your Flutter apps on the Android Virtual Machine. Happy coding!!!
