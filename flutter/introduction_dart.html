Question and Answers



What is Dart?

Dart is a programming language. Flutter uses Dart for cross platform app development.


Quick overview of Dart language:

Hello World

Every app has a main() function. To display text on the console, you can use the top-level print() function:

void main() { print('Hello, World!'); }

Variables

Most variables don't need explicit types:

var name = 'Voyager I'; var year = 1977; var antennaDiameter = 3.7; var flybyObjects = ['Jupiter', 'Saturn', 'Uranus', 'Neptune']; var image = { 'tags': ['saturn'], 'url': '//path/to/saturn.jpg' };

String data type

To store strings in dart or flutter, use String data type. Strings should either be enclosed in single quote or double quote.

String name = 'Robin Hood Pandey'; //String enclosed with single quote String city = "Bengaluru"; // String enclosed with double quote List<String> listOfApartments = ['Prestige Tranquility', 'Brigade Buena Vista', 'Brigade Golden Triangle' ]; // List of Strings

Control flow statements

Dart supports the usual control flow statements:

if (year >= 2001) { print('21st century'); } else if (year >= 1901) { print('20th century'); } for (var object in flybyObjects) { print(object); } for (int month = 1; month <= 12; month++) { print(month); } while (year < 2016) { year += 1; }

Functions

A function is a set of instructions to accomplish a task.

Example #1: A function which adds 2 numbers and displays the result:

void add() { int num1 = 10; int num2 = 20; int result = num1 + num2; print(result); }

Example #2: A function which receives 2 numbers as its arguments, adds them and displays the result:

void add(int num1, int num2) { int result = num1 + num2; print(result); }

Example #3: A function which receives 2 numbers as its arguments, adds them and returns the result:

int add(int num1, int num2) { int result = num1 + num2; return result; }

Example #4: A function which calls another function and receives the result:

int add(int num1, int num2) { int result = num1 + num2; return result; } void calculator() { int result = add(100, 300); print(result); }

Comments

Dart comments usually start with //.

// This is a normal, one-line comment. /// This is a documentation comment, used to document libraries, /// classes, and their members. Tools like IDEs and dartdoc treat /// doc comments specially. /* Comments like these are also supported. */

Imports

To access APIs defined in other libraries, use import.

// Importing core libraries import 'dart:math'; // Importing libraries from external packages import 'package:test/test.dart'; // Importing files import 'path/to/my_other_file.dart';
