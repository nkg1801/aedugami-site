Text Widget

A Text widget is used to display a string of text in flutter. By default, the text will be be displayed in one or multiple lines depending on the length of the text, or it can be made to display in a single line by applying layout constraints

Example 1: Display Simple Text

In the simplest form, a Text widget can be used like this:

Text('Hello World')

Full main.dart file

import 'package:flutter/material.dart'; void main() { runApp(MyApp()); } class MyApp extends StatelessWidget { @override Widget build(BuildContext context) { return MaterialApp( title: 'Flutter Demo', theme: ThemeData( primarySwatch: Colors.green, ), home: MyHomePage(title: 'My First Flutter app'), ); } } class MyHomePage extends StatefulWidget { MyHomePage({Key key, this.title}) : super(key: key); final String title; @override _MyHomePageState createState() => _MyHomePageState(); } class _MyHomePageState extends State<MyHomePage> { @override Widget build(BuildContext context) { return Scaffold( appBar: AppBar( title: Text(widget.title), ), body: Text('Hello World'), ); } }

Output


Example 2: Displaying text with some margins on the left, top, right, bottom, something like this:


To put margins around a widget, one can wrap the widget inside Container widget. Container widget has a margin property which can be used something like this:

Container( margin: EdgeInsets.all(30), child: Text('Hello World')), );

Full main.dart file

import 'package:flutter/material.dart'; void main() { runApp(MyApp()); } class MyApp extends StatelessWidget { @override Widget build(BuildContext context) { return MaterialApp( title: 'Flutter Demo', theme: ThemeData( primarySwatch: Colors.green, ), home: MyHomePage(title: 'My First Flutter app'), ); } } class MyHomePage extends StatefulWidget { MyHomePage({Key key, this.title}) : super(key: key); final String title; @override _MyHomePageState createState() => _MyHomePageState(); } class _MyHomePageState extends State { @override Widget build(BuildContext context) { return Scaffold( appBar: AppBar( title: Text(widget.title)), body: Container( margin: EdgeInsets.all(30), child: Text('Hello World')), ); } }

Output



To give a margin all around, use margin: EdgeInsets.all(margin_value), eg.

Container( margin: EdgeInsets.all(30), child: Text('Hello World')), );



To give margin only to specified direction, use only method with named parameters like left, right, top, bottom, something like, margin: EdgeInsets.only(left: margin_value, top: margin_value), eg.

Container( margin: EdgeInsets.only(left: 20, top:20), child: Text('Hello World')), );

Example 3: Text widget with different styling in one program:

The following example shows 5 Text widgets, wrapped inside Container and then all the Containers wrapped inside a Column widget. The reason for wrapping all Containers inside Column is because the body can take only one container. Also note, the children property of the Column widget. Since a Column can hold multiple widgets, it has children property, where as Container widget can hold only one element and hence its property is named child

@override Widget build(BuildContext context) { return Scaffold( appBar: AppBar(title: Text(widget.title)), body: Column(children: [ Container( margin: EdgeInsets.all(10), child: Text('A big red text', style: TextStyle(fontSize: 40, color: Colors.red)) ), Container( margin: EdgeInsets.only(top:20, left:10), child: Text('A big red text with italics', style: TextStyle(fontSize: 40, color: Colors.red, fontStyle: FontStyle.italic)) ), Container( margin: EdgeInsets.only(top:20, left:10), child: Text('A big text: Hello World', style: TextStyle(fontSize: 40, color: Colors.red, fontWeight: FontWeight.bold, fontStyle: FontStyle.italic)) ), Container( margin: EdgeInsets.only(top:20, left:10), child: Text('A multiline text without overflow: The quick brown fox jumped over the lazy sleeping dog', style: TextStyle(fontSize: 20, color: Colors.blue)) ), Container( margin: EdgeInsets.only(top:20, left:10), child: Text('A line of text with overflow: The quick brown fox jumped over the lazy sleeping dog', overflow: TextOverflow.ellipsis, style: TextStyle(fontSize: 20, color: Colors.blue)) ), ], ) ); }
