BottomNavigationBar Widget

A material widget that is displayed at the bottom of an app allowing the user to switch between small number of views. A BottomNavigationBar in android looks like the following:


Notable parameters:

  • type
  • items

type

type parameter of the BottomNavigationBar widget is used specify the type of the BottomNavigationBar. 2 values are possible: fixed and shifting

Usage: The following code snippet shows the usage of type parameter:

Scaffold( backgroundColor: Colors.green, appBar: AppBar(title: Text('I am learning flutter')), body: Text('My flutter app'), bottomNavigationBar: BottomNavigationBar( type: BottomNavigationBarType.fixed ), )



items

items parameter of the BottomNavigationBar widget is used to specify the list of BottomNavigationBarItem that needs to be displayed.

Usage:

Scaffold( backgroundColor: Colors.green, appBar: AppBar(title: Text('I am learning flutter')), body: Text('My flutter app'), bottomNavigationBar:BottomNavigationBar( type: BottomNavigationBarType.fixed , items: [ BottomNavigationBarItem( icon: Icon(Icons.home, color: Colors.black), label: 'Home' ), BottomNavigationBarItem( icon: Icon(Icons.add_shopping_cart, color: Colors.black), label: 'Car' ), BottomNavigationBarItem( icon: Icon(Icons.account_circle, color: Colors.black), label:'Profile' ), ], onTap: (index){ // index is the index of the BottomNavigationBarItem that is clicked // do what you want to do here when a particular item is clicked }, ) )


Complete main.dart demonstrating the usage of BottomNavigationBar. Just replace the content of your main.dart file with this and run the code.
